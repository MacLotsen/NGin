# game-engine
A C/C++ OpenGL Game Engine

## DEPENDENCIES
 - OpenGL2.1 (minimum)
 - [GLM](http://glm.g-truc.net/0.9.7/index.html)
 - [GLew](http://glew.sourceforge.net/)
 - [FreeGLut](http://freeglut.sourceforge.net/)
 - [FreeImage](http://freeimage.sourceforge.net/)

## INSTALLATION
```bash
cmake . && make
bin/preview
```
